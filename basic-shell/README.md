## C-Shell (Phase-1)  

This is a basic shell made in C.  

### Built-in commands  

* `echo`  
* `pwd`  
* `cd`  
* `pinfo`  
* `exit`  

### How to run the shell  

* In the directory that contains this README, run `make`.  
* Run the shell by typing, `./aaysh`.  
* Exit the shell by typing `exit` in the shell.
